# できること

VER-Lib は、VRChat のイベント活動に関わるみなさんの役割に合わせて、さまざまな機能を提供します。ここでは代表的なものを役割ごとに紹介します。各機能の詳しい使い方は、それぞれの章を参照してください。

## イベント運営ができること

イベントの告知から応募集計、抽選、当日のインスタンス割り振りまで、運営の一連の作業を自動化・効率化できます。

- **ワークフローでイベント運営を自動化できます** — 「フォーム投稿 → 応募締切 → 抽選 → 組分け → 通知」といったイベント進行を、ステップを組み合わせたワークフローとして組み立て、自動で実行できます。以下の機能の多くは、このワークフローの中の一手順として配置します。
- **定期メッセージを自動で送信できます** — 指定した時刻に、決まったメッセージを Discord チャンネルへ毎日自動で投稿できます。
- **キャスト・スタッフの参加を集計できます** — スタッフが登録した参加予定を、ロール（担当）ごとに取り込んで集計できます。
- **予約応募者を集計できます** — ゲストからの応募を集計できます。VRChat 上の Invite リクエストの集計や、外部 CSV からの取り込みに対応しています。
- **予約の抽選ができます** — 集めた応募者から、公平な抽選で当選・補欠・落選を決定できます。インスタンスごとの当選人数の設定や、連続して落選している人を優遇する救済措置、ブラックリスト・スタッフの除外にも対応しています。
- **インスタンスに割り振りできます** — 参加スタッフや当選ゲストを、定員・ロール条件を守りながら複数のワールドインスタンスへ割り振れます。自動での最適配置と、ドラッグ＆ドロップによる手動割り振りの両方が使えます。
- **メッセージ・Embed テンプレートを管理できます** — 告知や通知の文面テンプレート、Discord の Embed デザインを作成・プレビューできます。
- **Discord スレッドを自動で作成できます** — イベントごとに写真スレッドなどを自動作成し、スレッド名に開催回数やシリーズ名を差し込めます。
- **イベントシリーズを管理できます** — 定例イベントを「シリーズ」として定義し、インスタンス構成やロール、既定値をまとめて設定・継承できます。
- **イベントページ（告知ページ）を作成できます** — アイキャッチやキャスト一覧を載せた、参加者向けの告知ページを作成・編集できます。
- **グループやメンバーを管理できます** — メンバーの権限（オーナー / 管理者 / 編集 / 閲覧）管理、Discord などのアプリ連携設定、ブラックリスト登録、情報の公開範囲の設定ができます。

## スタッフができること

自分が参加するイベントの管理と、個人の予定管理をひとつのカレンダーでまとめて行えます。

- **参加登録できます** — 担当イベントごとに「参加する / 辞退する」を登録できます。同じ回に参加予定のスタッフ一覧も確認できます。
- **参加イベントの予定を一括で見れます** — 確定イベント・仮予定・自分の参加予約・個人予定を、月間 / 週間 / リストのカレンダーでまとめて確認できます。
- **外部カレンダーと同期できます** — 自分の予定を Google カレンダーなどに購読 URL（`.ics`）で同期できます。仮予定を含めるかも選べます。
- **個人の予定を登録できます** — 仕事のイベントとは別に、個人の予定（タイトル・日時・場所・色・タグ）を登録・編集できます。
- **予定をタグで分類できます** — 名前と色を付けたタグを作成し、予定の分類に再利用できます。
- **付箋でメモを残せます** — ダッシュボード上に付箋を貼ってメモを残せます。
- **グループごとのニックネームを設定できます** — 所属グループごとに、イベント参加時に使う表示名を設定できます。

## 参加者ができること

気になるイベントを見つけて応募し、募集開始や抽選結果を Discord で受け取れます。

- **イベントを探せます** — イベント一覧をキーワード検索・並び替えで探せます（ログイン不要）。Discord のスラッシュコマンドからも検索できます。
- **イベントを「いいね」できます** — 気になるイベントに「いいね（スキ）」を付けられます。いいねは、通知やカレンダー表示、予約の起点になります。
- **イベントに応募できます** — 開催回（セッション）ごとに応募・取消ができ、自分の応募状況を確認できます。
- **いいねしたイベントをカレンダーで見れます** — いいねしたイベントの開催予定を月間カレンダーで確認できます。
- **先行して参加予約できます** — 繰り返し開催されるイベントで、正式な応募開始前に将来の回へ参加意思を先行登録（予約）できます。
- **応募開始の通知を受け取れます** — 「いいね」したイベントの募集が始まると、Discord のダイレクトメッセージで通知が届きます。
- **当落の通知を受け取れます** — 応募したイベントの抽選結果（当選 / 落選）が、Discord のダイレクトメッセージで届きます。
- **おすすめのイベントを提案してもらえます** — Discord でおすすめのイベントを提案してもらえます。
- **イベント案内担当（Raspis）に相談できます** — Discord で案内担当のキャラクターに話しかけて、イベントについて相談できます。
- **イベントページを共有できます** — イベント詳細ページから、共有用の URL をコピーできます。
