# はじめる前に

VER-Lib はブラウザで使う Web サービスです。アプリのインストールは不要で、対応ブラウザからアクセスしてログインするだけで使い始められます。

## 必要なもの

- **Discord アカウント（必須）** — VER-Lib へのログインは Discord アカウントで行います。メールアドレスやパスワードによる登録はありません。
- **対応ブラウザ** — 一般的な最新の Web ブラウザ（Chrome / Edge / Safari / Firefox など）。
- **VRChat アカウント（任意）** — VRChat 連携を使いたい場合のみ必要です。登録そのものには必須ではありません。

## 立場によって追加で必要なもの

- **イベント運営** — Bot を招待できる Discord サーバー。グループ作成時に、この Discord サーバーと連携します。
- **スタッフ** — 特にありません（運営からの招待でグループに参加します）。
- **参加者** — 特にありません（ログインするだけで使えます。通知は Discord で届きます）。

準備ができたら、[ログイン](02_login.md) に進みます。
