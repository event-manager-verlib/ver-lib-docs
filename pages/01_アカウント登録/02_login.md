# ログイン（アカウント登録）

VER-Lib のログインは **Discord アカウント** で行います。専用の会員登録フォームはなく、初めてログインしたときにアカウントが自動で作成されます（メールアドレスやパスワードは使いません）。

## ログインの手順

1. VER-Lib のサイトを開き、**「Discord でログイン」** を選びます。
2. Discord の認証画面で連携を許可します。取得するのはユーザーの識別情報のみで、メールアドレスやパスワードを取得することはありません。
3. **初めてのログインの場合**、プライバシーポリシーの同意画面が表示されます。内容を確認して同意すると、利用を開始できます（同意が必要なのは初回の一度だけです）。
4. ログインが完了し、元の画面に戻ります。

```{note}
ログインは共通のポータルを経由して行われます。スタッフサイトやゲストサイトからログインした場合も、認証が終わると元のサイトに戻ります。
```

## サイトの入り口

VER-Lib には、立場ごとに次のサイトがあります。ログイン方法はいずれも同じ（Discord ログイン）です。

- **ポータル（マイページ）** — アカウント情報の確認・設定を行う場所です。ここから、自分が使うサイトを開けます。
- **イベント管理サイト** — イベント運営向け。
- **スタッフサイト** — スタッフ向け。
- **ゲストサイト** — 参加者向け。

ログイン後の設定は [アカウント設定（マイページ）](03_account_settings.md) を参照してください。
