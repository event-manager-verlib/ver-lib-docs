# アカウント設定（マイページ）

ログイン後は、**ポータルのマイページ** からアカウントに関する設定を行えます。

## マイページでできること

- **表示名の変更** — VER-Lib 上で表示される名前を変更できます（初期値は Discord の名前です）。
- **VER-Lib ID の確認** — アカウントごとに割り振られる ID を確認できます（変更はできません）。
- **Discord 連携の確認** — ログインに使っている Discord は「連携済み」として表示されます。
- **VRChat 連携** — VRChat アカウントを任意で連携できます（下記参照）。
- **退会** — アカウントを削除できます（下記参照）。

## VRChat 連携

VRChat アカウントを任意で連携できます。

- マイページの「VRChat を追加」から、自分の VRChat のユーザー ID（`usr_` で始まる形式）と表示名を登録します。
- 連携はいつでも解除できます。
- VRChat 連携は **本人確認のない自己申告制** です。ID は正確に入力してください。

## 退会とアカウントの復元

- 退会すると、アカウントは削除されます。
- 一度退会したあと、同じ Discord で再びログインすると、**「復旧する」**（いいねや予約などを引き継ぐ）か **「新しく作り直す」**（まっさらな状態で始める）かを選べます。

自分の立場に応じた最初の一歩は、クイックスタートの章（[運営編](../02_クイックスタート/01_operator.md) / [スタッフ編](../02_クイックスタート/02_staff.md) / [参加者編](../02_クイックスタート/03_participant.md)）を参照してください。
