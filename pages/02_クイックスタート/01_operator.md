# クイックスタート：イベント運営

イベント運営として、グループを作ってイベントを動かすまでの最初の一歩です。あらかじめ、Bot を招待できる Discord サーバーを用意しておいてください（→ [はじめる前に](../01_アカウント登録/01_prerequisites.md)）。

## 1. ログインする

イベント管理サイトを開き、「Discord でログイン」でログインします（→ [ログイン](../01_アカウント登録/02_login.md)）。

## 2. グループを作成する

「グループを作成」から新しいグループを作ります。作成した人が、そのグループのオーナーになります。

## 3. Discord サーバーを連携する

グループに Discord サーバーを連携し、Bot を招待します。告知や通知は、この Discord サーバーを通じて行われます。

## 4. メンバーを招待し、ロールを設定する

一緒に運営するスタッフをグループに招待し、担当ごとのロールを設定します。

## 5. ワークフローを作る

「告知 → 応募締切 → 抽選 → 割り振り → 通知」の流れをワークフローとして組み立てます。これで、イベント運営の一連の作業を自動化できます。

各機能の詳しい使い方は「イベント運営向け機能」の章を参照してください。
