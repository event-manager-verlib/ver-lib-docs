# クイックスタート：スタッフ

スタッフとして、グループに参加してイベントの参加登録を行うまでの最初の一歩です。

## 1. グループに参加する

運営からの招待でグループに参加します。方法は 2 通りあります。

- Discord に届く **招待リンク** からログインして参加する。
- 運営が、Discord サーバーのメンバーの中から直接あなたを追加する。

## 2. スタッフサイトにログインする

スタッフサイトを開き、「Discord でログイン」でログインします（→ [ログイン](../01_アカウント登録/02_login.md)）。

## 3. 参加するイベントに登録する

カレンダーから担当イベントを開き、「参加する」を登録します。辞退や取消もできます。

## 4. 予定をまとめて確認する

月間 / 週間 / リストのカレンダーで、参加予定・仮予定・個人予定をまとめて確認できます。

各機能の詳しい使い方は「スタッフ向け機能」の章を参照してください。
