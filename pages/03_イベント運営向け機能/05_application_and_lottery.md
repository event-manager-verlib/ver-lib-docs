# 応募の集計と抽選

## 応募者の集計

ゲストからの応募を集計できます。応募の入力元は複数に対応しています。

- **VRChat の Invite リクエスト** — VRChat 上のイベント参加申請（Invite）を集計します。
- **外部 CSV の取り込み** — 列の対応づけを指定して、CSV から応募者を登録します。

## 抽選

集めた応募者に対して、公平な抽選を実行し、当選・補欠・落選を決定します。

- **重み付き抽選** — インスタンスごとの当選人数（定員）を設定できます。
- **連続落選者への救済措置** — 過去に連続して落選しているユーザーの当選確率を優遇できます。
- **除外設定** — ブラックリストやスタッフを抽選から除外できます。
- **当落通知テンプレート** — 当選・落選それぞれの DM 文面を設定できます（未設定の場合は既定の文面が使われます）。

```{note}
抽選は実行すると取り消せません。実行前に内容を確認するようになっています。
```
