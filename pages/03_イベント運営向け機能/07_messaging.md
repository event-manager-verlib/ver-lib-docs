# メッセージ・通知

## メッセージテンプレート・Embed

告知や通知の文面を **テンプレート** として作成・管理できます。Discord の **Embed** デザインも作成・プレビューでき、変数を差し込めます。

## 定期メッセージ

指定した時刻に、決まったメッセージを Discord チャンネルへ自動で投稿できます。イベント個別ではなく、グループ単位の定期的なお知らせに使います。

```{note}
現在の定期メッセージは、**毎日（時刻指定）** の送信に対応しています。
```

## スレッドの自動作成

イベントごとに、Discord スレッド（写真スレッドなど）を自動で作成できます。スレッド名には、開催回数やシリーズ名などの変数を差し込めます。

## 締切集計とまとめ通知

運営が送ったメッセージを管理対象に登録しておくと、締切が来たときに反応状況（既読・未読など）を集計して、まとめて通知できます。
